# Phase 2.3 UAT Sign-Off Register

**Status:** Template / pending sign-off  
**Applies to:** AIM Tank Integrity MVP UAT Cycle 1  
**Rule:** No actual sign-off is claimed by this template.

## 1. Sign-Off Rule

No one may sign off if unresolved blocker, critical, or governance defect remains unless formally accepted with documented risk and owner approval.

Sign-off decisions allowed:

- approved;
- approved with conditions;
- rejected.

Any accepted risk must reference the defect ID, owner, mitigation, expiry/target date, and approving role.
