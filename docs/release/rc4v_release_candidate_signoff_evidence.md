# RC4-V Release Candidate Signoff Evidence

Purpose: capture human release-candidate signoff after target-environment validation.

| Role | Required | Evidence Required | Decision |
|---|---:|---|---|
| Product Owner | Yes | Go / Conditional Go / No-Go decision recorded | Pending |
| Engineering Lead | Yes | Release tag, API route, calculation governance, and OpenAPI checks validated | Pending |
| Platform/DevOps Lead | Yes | Deployment, rollback, backup/restore, monitoring, and alerting validated | Pending |
| Security Lead | Yes | RBAC, service actor boundary, audit log, and secrets checks validated | Pending |
| QA/UAT Lead | Yes | UAT evidence, smoke tests, and open defect disposition validated | Pending |

## Final Decision

Final decision: Go / Conditional Go / No-Go

Conditional Go must list conditions, owners, deadlines, and rollback triggers.
